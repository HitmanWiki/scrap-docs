Transfers

GHOSTwire allows you to transfer SOL and tokens between your own wallets or to external addresses. All transfers are executed directly on Solana blockchain.

Transfer Types Available

• SOL: Between My Wallets — Move SOL between W1, W2, W3, etc.
• SOL: To External — Send SOL to any Solana address
• Token: Between My Wallets — Move tokens between your wallets
• Token: To External — Send tokens to any Solana address

Transfer Menu

Click Transfers from the main menu to access all transfer options.

Important Notes

⛽
Gas Fees
SOL transfers require ~0.000005 SOL for gas. When sending "all", bot automatically reserves gas.
🪙
Token Decimals
Bot auto-detects token decimals (6, 9, etc.) for accurate transfers.
📝
Minimum Transfer
Minimum transfer is 0.000001 SOL (1000 lamports). Smaller amounts will be rejected.
🔄
Position Updates
After transferring tokens, click Positions → Refresh to update balances.

Amount Input Format

0.1 → Transfer 0.1 SOL/tokens
all → Transfer entire balance (minus gas for SOL)
cancel → Cancel the transfer

Troubleshooting

"Insufficient balance" error

Make sure you have enough SOL for the transfer + gas fees (~0.000005 SOL). Try sending a smaller amount or use "all" to send everything minus gas.

"Invalid address" error

Solana addresses are base58 encoded and 32-44 characters long. Double-check the address you're sending to.

Token transfer failed

Make sure the destination wallet has an Associated Token Account (ATA) for that token. The bot creates it automatically if needed.

Transfers FAQ

How long do transfers take?

Solana transfers complete in 2-5 seconds typically. The bot shows the TXID immediately.

Can I cancel a transfer?

No, blockchain transactions are irreversible. Always double-check addresses and amounts before confirming.

Do I pay fees for transfers?

Only standard Solana gas fees (~0.000005 SOL). No additional bot fees for transfers.

What happens to my positions after transferring tokens?

Tokens transferred to another wallet will no longer show in the source wallet's positions. Click Refresh to update.
