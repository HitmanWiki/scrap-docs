How P&L is Calculated

📊
For Active Positions
P&L % = (Current Price - Entry Price) / Entry Price × 100. Entry price recorded from buy transaction.
📉
For Closed Trades
P&L SOL = (Exit Price × Amount) - (Entry Price × Amount). Both prices from on-chain data.
🔄
Real-Time Prices
Prices fetched from Jupiter API and DexScreener for accurate current valuations.

Position Sync Logic

GHOSTwire always checks your actual wallet balance on-chain:

  • Checks Associated Token Account (ATA) for each token
  • If balance is 0 → position is closed automatically
  • If balance decreased → amount updated (partial sell)
  • If balance increased → amount updated (additional buy)

Portfolio FAQ

Why doesn't my position show up immediately?

Positions are saved immediately, but the pinned message refreshes every 5 minutes. Click Refresh to see it instantly.

How is total portfolio value calculated?

Total = SOL balance + (sum of all token amounts × current price). Prices from Jupiter API and DexScreener.

Can I see P&L for tokens bought outside GHOSTwire?

Yes! Use Refresh to sync your wallet. The bot detects all tokens in your wallet.

What happens to positions when I sell partially?

The bot updates the position amount. You'll see reduced balance and sell recorded in trade history.
