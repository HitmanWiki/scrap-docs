Buying & Selling

GHOSTwire supports both manual trading and automated sniping. Buy any Solana token instantly, sell with custom amounts, and let auto-sell handle your take profits.

Fee System

Each trade has a 0.05% fee. Whitelisted users pay 0%. Fees are automatically collected and distributed to referrers (20% commission).

Auto-Snipe: When Channels Trigger Buys

This is the core feature of GHOSTwire. Once you add a channel, the bot monitors it 24/7 and automatically buys when a token contract is detected.

How Auto-Snipe Works
📡 Channel Message → 🔍 Extract CA → ✅ Check Holding → 💰 Check Balance → 🔥 Execute Buy → 📨 Send Notification

Public Channels

Any public Telegram channel can be added. The bot uses a dedicated monitor account to read messages.

Public Channel Flow:

  • Add channel via Add Channel → Public
  • Bot joins the channel automatically
  • Every new message is scanned for Solana addresses
  • When CA found → auto-snipe triggers
Note

Private Channels

Private channels require you to authenticate your Telegram account first (QR or OTP). The bot then monitors using YOUR session.

Private Channel Prerequisites:

  • Complete TG Auth (QR code or OTP)
  • Add channel via Add Channel → Private
  • Bot uses your session to read the private channel
  • Same auto-snipe logic applies
Important

You must be a member of the private channel. Bot cannot join private channels automatically.

Auto-Snipe Settings

🔫
Global Auto-Buy Toggle
Main Menu → Settings → Auto-Buy. Turn OFF to stop all auto-snipes while keeping channels added.
📡
Per-Channel Wallet
Each channel can use a different wallet. W1 for conservative signals, W2 for high-risk memes.
📊
Per-Channel Slippage
Set higher slippage (15-20%) for volatile meme coin channels. Lower (5-10%) for established tokens.
🎯
Per-Channel Take Profit
Different profit targets per signal source. Meme channels: 100%+. Blue chip channels: 30%.
Anti-Duplicate Protection

GHOSTwire prevents buying the same token twice within 48 hours. This protects you from FOMO buys after price pumps and multiple channels posting the same signal.

⏭️ Already holding GNL7MQSz... Bought 2.5 hours ago. Skipping buy.
Daily Trade Limit

You can set a maximum number of auto-trades per day (default: 100). Once reached, auto-snipe pauses until next day. Manual buys are still allowed.

Trading FAQ

Why did my buy fail with "0 tokens"?

Some tokens have unusual decimals. GHOSTwire auto-detects decimals but may need a retry. Increase slippage for volatile tokens.

How is the token amount calculated?

Bot fetches the actual transaction from Solana to get the exact tokens received — not just the quote.

What happens if a sell fails?

Bot verifies the transaction on-chain. If failed, your tokens remain safe and you can retry with smaller amount or higher slippage.

Can I sell tokens not bought through GHOSTwire?

Yes! Use Sell by Address — enter any token address you hold, and GHOSTwire will sell your balance.

Why didn't the bot auto-snipe a signal?

Check: 1) Auto-Buy is ON in Settings, 2) Daily limit not reached, 3) Not already holding that token (48h cooldown), 4) Sufficient SOL balance.
