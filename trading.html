Buying & Selling

GHOSTwire supports both manual trading and automated sniping. Buy any Solana token instantly, sell with custom amounts, and let auto-sell handle your take profits.

Fee System

Each trade has a 0.5% fee. Whitelisted users pay 0%. Fees are automatically collected and distributed to referrers (20% commission).

Auto-Snipe: When Channels Trigger Buys

This is the core feature of GHOSTwire. Once you add a channel, the bot monitors it 24/7 and automatically buys when a token contract is detected.

How Auto-Snipe Works
📡 Channel Message→🔍 Extract CA→✅ Check Holding→💰 Check Balance→🔥 Execute Buy→📨 Send Notification

Public Channels

Any public Telegram channel can be added. The bot uses a dedicated monitor account to read messages.

Public Channel Flow:

  • Add channel via Add Channel → Public
  • Bot joins the channel automatically
  • Every new message is scanned for Solana addresses
  • When CA found → auto-snipe triggers
Note

Public channels work immediately. No authentication needed.

Private Channels

Private channels require you to authenticate your Telegram account first (QR or OTP). The bot then monitors using YOUR session.

Private Channel Prerequisites:

  • Complete TG Auth (QR code or OTP)
  • Add channel via Add Channel → Private
  • Bot uses your session to read the private channel
  • Same auto-snipe logic applies
Important

You must be a member of the private channel.

Trading FAQ

Why did my buy fail with "0 tokens"?

Some tokens have unusual decimals. Increase slippage for volatile tokens.

How is the token amount calculated?

Bot fetches actual transaction from Solana to get exact tokens received.

What happens if a sell fails?

Tokens remain safe. Retry with smaller amount or higher slippage.

Can I sell tokens not bought through GHOSTwire?

Yes! Use Sell by Address — enter any token address you hold.

Why didn't the bot auto-snipe a signal?

Check: Auto-Buy ON, daily limit, 48h cooldown, sufficient SOL balance.
