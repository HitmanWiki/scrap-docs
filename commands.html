📋 Basic Commands

Command Description
/start Start the bot and show main menu
/help Display help information
/cancel Cancel current operation and return to main menu

📈 Trading Commands

Command Description
/balance Check SOL balance across all wallets
/positions View all active positions with P&L
/channels List all monitored channels
/removechannel Remove a monitored channel

🔗 Referral Commands

Command Description
/referral Show your referral link and earnings stats

🔒 Private Channel Commands

Command Description
/setapi ID HASH PHONE Set Telegram API credentials for private channels
📌 Example: /setapi 12345678 abc123def +1234567890

💡 Tips

  • Most features are accessible via the inline menu buttons — no need to memorize commands!
  • Type /cancel anytime to exit the current operation
  • Commands are case-insensitive — /START works the same as /start
  • Use the search box above to find specific features
